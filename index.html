Tips & Notes

  • You can right-click on any package link and choose "Save link as…" to download the file manually—especially useful for theme files like .tres that open in browser by default.
  • Packages labeled "View Resource File" are usually text-based and may open directly in your browser. Use "Save link as…" to store them locally.
  • All packages are compatible with Text Forge version 0.1.0 and above unless stated otherwise.
  • Official packages are maintained by the Text Forge team. Community contributions are welcome via pull requests.
  • To install a package manually, download the file and place it inside your Text Forge/packages/ directory.
