Tips & Notes

  • You can right-click on any package link and choose "Save link as…" to download the file manually—especially useful for theme files like .tres that open in browser by default.
  • Packages labeled "View Theme File" are usually text-based and may open directly in your browser. Use "Save link as…" to store them locally.
  • Official packages are maintained by the Text Forge team. Community contributions are welcome via pull requests.
  • To install a theme manually, download the file and place it inside Editor Data/themes/ directory. (You can find editor data directory with Settings > Open Data Folder)
  • To install a mode manually, download the file and import it with Settings > Mode Manager > Import Mode / Mode Kit.
  • To install an extension manually, download the file and install it with Settings > Extensions > Install Extension.
