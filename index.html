Tips & Notes

Download Tips

Right-click on any package link and choose "Save link as..." to download manually. Especially useful for theme files like .tres that open in browser by default.

Official Packages

Official packages are maintained by the Text Forge team. Community contributions are welcome via pull requests.

Installing Themes

Download the file and place it inside Editor Data/themes/ directory. Find it via Settings > Open Data Folder.

Installing Modes

Download the file and import it with Settings > Mode Manager > Import Mode / Mode Kit.

Installing Extensions

Download the file and install it with Settings > Extensions > Install Extension.
